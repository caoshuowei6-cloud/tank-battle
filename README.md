# 🎮 坦克大战 - Tank Battle

一个使用纯 HTML5 Canvas + JavaScript 制作的经典坦克大战游戏。

## 🌟 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏！

👉 [点击在线试玩](https://你的用户名.github.io/tank-battle/) （部署到GitHub Pages后可用）

## 📸 游戏截图

*经典街机风格的坦克大战，支持多关卡、多种敌人*

## 🎯 游戏特性

- ✅ **经典玩法** - 还原街机版坦克大战核心体验
- ✅ **多种敌人** - 普通坦克、快速坦克、重型坦克
- ✅ **多关卡系统** - 砖墙、钢墙、水域等地形变化
- ✅ **完整的碰撞检测** - 子弹、坦克、地形全面碰撞
- ✅ **粒子爆炸效果** - 击中敌人时的酷炫爆炸动画
- ✅ **基地保护** - 保护你的基地不被摧毁
- ✅ **计分系统** - 消灭不同敌人获得不同分数
- ✅ **关卡进度** - 完成一关后进入下一关

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` | 发射子弹 |
| `P` | 暂停/继续游戏 |

## 🧱 地形说明

| 地形 | 说明 |
|------|------|
| 🔴 砖墙 | 可以被子弹破坏 |
| ⚪ 钢墙 | 坚不可摧 |
| 🔵 水域 | 坦克无法通过 |
| 🟢 草地 | 隐藏坦克（待实现） |
| ⭐ 基地 | 保护它！被击中游戏结束 |

## 👾 敌人类型

| 类型 | 颜色 | 速度 | 生命值 | 分数 |
|------|------|------|--------|------|
| 普通坦克 | 红色 | 慢 | 1 | 100 |
| 快速坦克 | 青色 | 快 | 1 | 200 |
| 重型坦克 | 紫色 | 很慢 | 3 | 300 |

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/tank-battle.git

# 进入目录
cd tank-battle

# 直接用浏览器打开
open index.html
```

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# 然后访问 http://localhost:8080
```
